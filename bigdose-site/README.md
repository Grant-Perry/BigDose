# BigDose Marketing Site

Landing page for [BigDose](https://bigdose.web.app) — vitamin D tracking and skin safety for iPhone.

## Develop

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Firebase project: `bigdose` → `https://bigdose.web.app`
